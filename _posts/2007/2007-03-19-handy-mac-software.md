A few days onwards with my spiffy new macbook, I've discovered a lot of useful tools:

Freeware apps:

* [Adium](http://web.archive.org/web/20070314003534/http://adiumx.com:80/)[^ia6]: IM client
* [Chmox](https://chmox.sourceforge.net/): CHM Viewer
* [ClearDock](http://web.archive.org/web/20070322043503/http://www.unsanity.com:80/haxies/cleardock)[^ia7]: changes/removes your dock background
* [coconutBattery](https://coconut-flavour.com/coconutbattery/index.html): extensive battery information
* [coconutWiFi](https://www.coconut-flavour.com/coconutbattery/): detects (open) wifi networks and shows it in the menu bar
* [Growl!](https://growl.github.io/growl/): notification system
* [iCyclone](http://www.alitacrew.org/icyclone/): fan controller
* [iStumbler](http://www.istumbler.net/): network stumbling tool
* [iTerm](https://iterm.sourceforge.net/): the terminal redefined
* [KisMAC](https://web.archive.org/web/20060208021046/http://kismac.de:80/)[^ia8]: Wireless Network Stumbler
* [Monolingual](https://sourceforge.net/projects/monolingual/): Delete unneccesary languages
* [NeoOffice](https://neooffice.github.io/): Office suite for Mac (based on OpenOffice)
* [OnyX](http://web.archive.org/web/20070317005118/http://www.titanium.free.fr:80/pgs2/english/onyx.html)[^ia9]: System maintenance
* [Remote Desktop Connection](http://web.archive.org/web/20070331061447/http://www.microsoft.com:80/mac/otherproducts/otherproducts.aspx?pid=remotedesktopclient)[^ia1]: RDP client (for Windows Terminal Services)
* [Smcfancontrol](http://web.archive.org/web/20070316154308/http://www.conscius.de:80/~eidac/index.html)[^ia10]: temperature menulet and fan control
* [Sidenote](https://web.archive.org/web/20060116154627/http://www.apple.com:80/downloads/macosx/productivity_tools/sidenote.html)[^ia2]: notes
* [StuffIT! Expander](http://web.archive.org/web/20070323041252/http://www.stuffit.com:80/mac/expander/index.html)[^ia11]: archive extractor
* [The Unarchiver](https://theunarchiver.com/): archive extractor
* [uApp](http://www.eagle-of-liberty.com/uapp/index_en.php): uninstaller (removes prefs, ... too)
* [UnrarX](https://web.archive.org/web/20060704233247/http://www.unrarx.com:80/)[^ia12]: RAR extractor
* [VLC media player](http://www.videolan.org/vlc/): media player
* [witch](https://manytricks.com/witch/): Window switcher (not just app switching)
* [Xfolders](http://www.kai-heitkamp.com/cms_en/main.php?content=9&module=0&SprachID=1&uid=d96f9e3704277b05bb3345e167aa7afe): file manager
* [XMeeting](https://xmeeting.sourceforge.net/): SIP/H.323 video conferencing client

Non-free software:

* [MenuCalendarClock](http://www.objectpark.net/mcc.html): unfolding menulet calendar with iCal integration
* [Parallels](https://www.parallels.com/): Virtualisation software
* [Roxio Toast](https://web.archive.org/web/20060716063109/http://www.roxio.com:80/enu/products/toast/titanium/overview.html)[^ia3]: CD/DVD burning
* [Transmit](https://www.panic.com/transmit/): FTP/SCP client
* [Unison](https://panic.com/blog/the-future-of-unison/): Usenet (NNTP) client

Plugins:

* [Flip4mac WMV](https://web.archive.org/web/20060101050327/http://www.flip4mac.com:80/wmv.htm)[^ia13]: wmv plugin for QuickTime

Preferences:

* [StartupSound.prefPane](http://web.archive.org/web/20070314031854/http://macupdate.com:80/info.php/id/16425)[^ia4]: to get rid of that (annoying?) sound when you start your Mac

Widgets:

* [App Update](https://web.archive.org/web/20060426034345/http://www.dashboardwidgets.com:80/showcase/details.php?wid=1510)[^ia14]: shows if there are updates for apps
* [iStat Pro](http://web.archive.org/web/20070315041642/http://www.apple.com:80/downloads/dashboard/status/istatpro.html)[^ia5]: hardware stats

If I find more, I'll list them here ;)
